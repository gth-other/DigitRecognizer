# Нейронная сеть, распознающая рукописные цифры
Используемая нейронная сеть является многослойным перцептроном.<br><br>
Нейронная сеть уже обучена и ее данные хранятся в каталоге **../data/DigitRecognizer.cfg**, но функционал для самостоятельного обучения оставлен.<br><br>
В репозитории содержится **70 000 JPG** изображений в разрешении **28x28**, причем **60 000** изображений используются для обучения нейронной сети, а оставшиеся **10 000** - для проверки качества нейронной сети. Изображения были составлены при помощи библиотеки **MNIST** и хранятся в каталоге **../data/images**.<br><br>
Дальнейшее изучение репозитория рекомендуется начать с файла **../src/main.cpp**.
